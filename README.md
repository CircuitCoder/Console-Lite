## 开发
在 Clone 项目过后，请使用以下指令安装依赖:

```bash
npm install
npm run rebuildNative # Rebuild native modules for Electron
```

启动应用：

```bash
npm start
```

启动服务器：

```bash
npm run server 
```

生成可执行文件：

```bash
npm prune --producation # 删除开发依赖
npm run pack
```
